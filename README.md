# Algo Arena Backend

Backend system for **Algo Arena**, an online competitive programming platform where users can solve coding problems, participate in contests, and receive automatic code evaluation.

## Features

- User authentication and account management
- Problem creation and management system
- Contest creation and participation
- Code submission and automatic judging
- Queue-based code execution system
- Support for multiple programming languages
- Scalable backend architecture

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Redis / Queue System
- Docker (for secure code execution)

## Project Structure

```
controllers/
routes/
models/
workers/
queues/
utils/
scripts/
```

## Installation

Clone the repository:

```
git clone https://github.com/UtkarshPratapSingh7777/Algo-Arena.git
```

Install dependencies:

```
npm install
```

Run the server:

```
npm run dev
```
